Explicación implementación:

En el proyecto adjunto estan las soluciones para cada uno de los problemas escogidos, por medio de la consola de java se debe realizar la interacción con las soluciones.

Para este proyecto se requirieron 6 clases:

	- Edge.java: contiene el modelado de un eje de un grafo, con nodo de incio, nodo de fin y costo.
	- Vertex.java: contiene el modelado de un vertice de un grafo, con sus hijos y el identificador.
	- ejecutable.java: Contiene el codigo general y la interacción con la consola de las distintas soluciones.
	- Problema1.java : Contiene la implementacion de la solucion al problema 1 (Grado 6 de amistad). Se incluyen 3 casos de prueba, 
			   en uno se cumple la teoría de los 6 grado y en los otros: existe un nodo aislado y hay mas de 7 grados de conexión.
	- Problema2.java : Contiene la solución para el problema del autoprestamos de los bancos. Hay dos casos de prueba, en uno hay autoprestamos entre bancos
			   en el otro no hay autoprestamos.
	- Problema3.java : Contiene la solucion para el problema de las calles en la ciudad, el mapa de nodos y ejes se encuentra anexo como archivo de nombre
			   "mapa.png".
